# td

To-Do ToDay is a simple TUI app for tracking tasks with a focus on daily workflow. Adding and checking tasks, while the backend
storage is a directory of markdown files.

Usage:
```
td --help
```

# run locally
```bash
go run main.go
```

# test
```bash
go test -v ./...
```

# build

```bash
go build -o td main.go
```
